## これはなに
dff素子, nand素子, バスのみを所与のビルドブロックとして、その上にハードウェア / OSを構築し、またレイヤー間のつなぎとしてアセンブラ、VM、コンパイラを言語とともに実装。


- ハードウェア
    - 各種論理ゲート
    - cpu: ALU, decoder, その他comp,dest,jumpロジック処理コンポーネント, register
    - memory: cell, word, memory
- アセンブラ
- VM
- 高水準言語とそのコンパイラ
- OS
